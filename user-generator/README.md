# RandomUser

## Stack usada
- VueJs.
- API https://randomuser.me/api


### Clone o repositório
$ git clone https://github.com/add-flav1s/user-generator.git


### Entre no repositório
$ cd user-generator


### Instale as dependências
$ npm install


### Execute a aplicação
$ npm run serve
